# GitHub Explorer

## Project Overview

GitHub Explorer is a Web Technology project that allows users to search and explore GitHub profiles and repositories using the GitHub REST API.

The application provides a user-friendly interface to search GitHub usernames, view complete profile information, explore repositories, and navigate directly to GitHub profile and repository pages.

## Main Features

- GitHub profile search
- Display user profile details
- View followers, following, repositories, bio, and location
- Explore user repositories
- Global repository search
- Display repository information
- Redirect to original GitHub profile and repository pages

## Technologies Used

- Java
- JSP
- Servlets
- Maven
- Apache Tomcat
- HTML
- CSS
- JavaScript
- GitHub REST API

## Project Objective

The objective of GitHub Explorer is to develop a web application that integrates external APIs and provides dynamic GitHub data through an interactive interface.

## Conclusion

GitHub Explorer demonstrates the implementation of web technologies, API integration, and dynamic data handling to create an efficient platform for exploring GitHub users and repositories.
